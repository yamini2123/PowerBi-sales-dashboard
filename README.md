# Automated Sales Analytics & Reporting Dashboard

An interactive Business Intelligence dashboard built using Power BI and Python for automated data cleaning and reporting.

## Project Overview

This project focuses on analyzing sales performance, profit trends, regional insights, and customer behavior using interactive visualizations in Power BI.

The dataset was cleaned and transformed using Python before creating the dashboard in Power BI.

## Features

* Sales Performance Analysis
* Profit & Revenue Insights
* Regional Sales Comparison
* Interactive Filters & Slicers
* Automated Data Cleaning Workflow
* Dynamic Visualizations

## Tools & Technologies

* Power BI
* Python
* Power Query
* DAX
* Kaggle Dataset
